Portfolio Work

BotHop University Project

For my Major Project I was tasked with creating a video game, which led me to create a 3D Puzzle Game in which the Player must take control of a variety of different Robots with different abilities. This is done by using an ability known as BotHop, which allows any Robot to hop consciousness into another Robot so long as they can make direct line of sight.

Crime Scene Recreation College Project

In this assignment, I was tasked with recreating an "Crime Scene" interactive Crime Scene using 3DS Max and Unreal Engine 4, and is one of my favorite pieces of work so far.

The Prisoner Chess University Project

This assignment was to make a game for a convention related to "The Prisoner", and so I chose chess as it provides a two player experience in a convention full of single player projects. The game had abilities to differentiate it, each being able to change the flow of the game and spawning every 10 turns in game.

Environmental Experimentation College Project

This assignment was to make some household assets, which could then be used in a game.

My First Max College Project

My first time in 3D I decided to make an animation that takes place in a cel shades Desert, followed by a fly-through of an environment that ends with a Robot being constructed.

Extinction Pixel Art Game College Project

Extinction was a 2D Pixel Art game made in GameMaker, and had the Player controlling a Planktalien during an invasion of an underground facility.
